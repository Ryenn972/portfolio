# Portfolio – PHP MVC

**Live demo** :  
https://ryennmithcatan.sites.3wa.io/ue6/mvc/portfolio/index.php

Ce projet est un **portfolio personnel dynamique** développé en **PHP** selon une **architecture MVC**, permettant de présenter des projets, des compétences et de gérer le contenu via un **dashboard administrateur**.

---

## Fonctionnalités

### Utilisateur
- Affichage des informations personnelles
- Description dynamique
- Photo de profil
- Page de détails du profil

### Projets
- Liste de projets
- Détails d’un projet
- Catégories associées
- Labels (tags) multiples
- Lien externe vers le projet

### Dashboard (Admin)
- Modification du profil
- Ajout / modification / suppression de projets
- Upload d’images
- Sélection de catégories
- Labels multi-sélection (checkbox)
- Interface responsive (mobile first)
